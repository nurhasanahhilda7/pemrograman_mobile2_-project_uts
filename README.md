Fitur Aplikasi
Aplikasi ini memiliki fitur navigasi dan header yang menampilkan judul serta tombol Order Now. Tersedia pencarian dan filter kategori untuk memudahkan pengguna menemukan menu yang diinginkan. Katalog menu ditampilkan dalam bentuk card berisi gambar, nama, harga, dan tombol favorit. Panel favorit menampilkan jumlah serta daftar menu yang sudah dipilih pengguna. Aplikasi juga menyediakan review pelanggan berupa nama, komentar, dan rating. Seluruh tampilan dibuat modern, responsif, dan menyesuaikan ukuran layar.
